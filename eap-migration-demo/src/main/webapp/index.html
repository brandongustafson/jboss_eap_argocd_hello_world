EAP Migration Demo

This app runs on both JBoss EAP 6.4 and 7.4 and demonstrates key migration points.

Endpoints

Servlet: /demo/hello?name=Customer REST: /demo/api/greet?name=Customer

Key Migration Points

  • Web Container: JBoss Web (Tomcat-based) → Undertow
  • Java EE: 6 → 7
  • Servlet: 3.0 → 3.1
  • JAX-RS: 1.1 → 2.0 (RESTEasy 2.x → 3.x)
  • CDI: 1.0 → 1.2 (bean-discovery-mode changes)
  • JPA: 2.0 → 2.1
  • EJB: 3.1 → 3.2
